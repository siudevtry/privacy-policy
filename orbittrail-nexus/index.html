Privacy Policy for Orbittrail Nexus

This Privacy Policy explains how the Orbittrail Nexus browser extension handles information when you install and use it.

Last updated: July 13, 2026

◎
No personal data collection The extension does not collect names, email addresses, browsing history, or personal identifiers.
⌁
Local settings only Preferences are stored locally in the user's browser through the Chrome storage API.
✓
No advertising or tracking The extension does not display ads, track users, or use analytics services.

1 Overview

Orbittrail Nexus is a visual browser extension that creates glowing orbs around the cursor, with trails that gradually fade over time.

The effect is activated by the user and can be customized through the extension's toolbar popup.

2 Information we collect

Orbittrail Nexus does not collect, record, transmit, sell, or share personal information.

  • No names or email addresses are collected.
  • No browsing history is collected.
  • No website content is stored or transmitted.
  • No search queries or form entries are collected.
  • No location information is collected.
  • No authentication or financial information is collected.
  • No analytics or behavioral tracking is used.

3 Local storage

The extension uses the browser's local storage functionality only to remember user-selected preferences, such as whether the effect is enabled and how the visual effect is configured.

These preferences remain inside the user's browser and are not sent to the developer or to an external server.

4 Permissions

Orbittrail Nexus requests only the permissions needed to provide its declared visual effect.

activeTab

Used only after the user invokes the extension, allowing the effect to run on the currently active browser tab.

scripting

Used to insert and run the extension's visual-effect code on the active tab after the user initiates the action.

storage

Used to save extension preferences locally in the user's browser so the selected settings can be restored later.

These permissions are not used to monitor browsing activity, collect page content, or build a user profile.

5 Remote code and external services

Orbittrail Nexus does not download or execute remote code. All executable extension code is included within the extension package distributed through the browser extension store.

The extension does not use external analytics platforms, advertising networks, tracking pixels, or third-party data collection services.

6 Data sharing and selling

Because Orbittrail Nexus does not collect personal data, there is no personal data to sell, rent, disclose, or share with third parties.

7 Data security

User preferences are handled through the browser's extension storage system. The extension is designed to keep its operation limited to the functionality described in this policy.

8 Children's privacy

Orbittrail Nexus does not knowingly collect personal information from children or from any other users.

9 Changes to this policy

This Privacy Policy may be updated if the extension's functionality or data-handling practices change. The latest version will always be published on this page, together with its revision date.

10 Contact

For questions about this Privacy Policy or the extension's privacy practices, contact the developer through the project's GitHub issue page.
